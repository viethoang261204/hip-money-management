Money Manager (Quản lý Nợ & Vay)

Chạy dự án:

1. Cài đặt phụ thuộc
```
npm install
```

2. Chạy chế độ dev (tự reload)
```
npm run dev
```

3. Truy cập
```
http://localhost:3000
```

Cấu trúc chính: `Person` (người), `Loan` (khoản vay/nợ), `Payment` (thanh toán). CRUD đầy đủ, giao diện EJS + Bootstrap, dữ liệu SQLite file `data.sqlite`.
